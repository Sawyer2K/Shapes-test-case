Приложение для расчёта параметров фигуры по заданным данным и вывода их в консоль или записи в файл.

Источник входных данных (указывается обязательно) определяется аргументом командной строки при запуске программы: -file [pathToInputFile]
Источник выходных данных определяется аргументом командной строки при запуске программы: -out [pathToOutputFile]
Если источник выходных данных не указан, информация будет выведена в консоль.

Программа считывает данные из файла. Первая строка – тип фигуры, вторая строка – параметры фигуры. Поддерживаются только целочисленные параметры! 
Разрешенные типы фигур и необходимые параметры для них:
CIRCLE – радиус
RECTANGLE – длина, ширина
TRIANGE – сторона А, сторона В, сторона С

Вся интеракция со стороны пользователя происходит только во время запуска программы, поэтому при возникновении даже обрабатываемых ошибок в процессе работы, бросаются исключения и программа завершается с соответствующими пояснениями (я понимаю, что это наверно не нормальная практика в коммерческих приложениях).

Проект собран в Gradle, unit-тесты на junit 5, логгирование на slf4j-log4j.
